# GitHub References for タマネギアプリ Token/SBT & UX

- **ethereum/EIPs** — ERC-5192 (Minimal Soulbound NFT) reference implementation; useful for non-transferable attestation primitives.  
  https://github.com/ethereum/EIPs/tree/master/EIPS/eip-5192

- **masa-finance/erc20-soulbound-token** — Sample contracts for ERC20-style soulbound tokens; shows non-transferable fungible pattern.  
  https://github.com/masa-finance/erc20-soulbound-token

- **spruceid/ssi** — Decentralized identity & verifiable credentials toolkit in Rust; can back selective disclosure for SBT metadata.  
  https://github.com/spruceid/ssi

- **lens-protocol/core** — On-chain social graph with gated publications;参考になるソーシャル×トークン設計。  
  https://github.com/lens-protocol/core

- **destiner/awesome-soulbound-tokens** — Curated list of SBT contracts, papers, and tooling;速く関連実装を横断把握できる。  
  https://github.com/destiner/awesome-soulbound-tokens

- **hoprnet/hoprnet** — Privacy-preserving incentivized messaging;インセンティブとプライバシー両立の設計参考。  
  https://github.com/hoprnet/hoprnet
