# 概要
`output/タマネギアプリ/UX/ユーザー体験フロー.mmd`は、TypeScriptモジュールで定義したユーザー体験フローをMermaidアクティビティ図として出力したファイル。

# 目的
- UML図の可視化用Mermaidコードをリポジトリ内で共有し、ドキュメントビューアやMermaid対応ツールから即表示できるようにする。

# 最適な実装方法
1. `toMermaidActivityDiagram`で生成したコードを`.mmd`ファイルに保存。
2. Markdownやドキュメントツールから参照できるよう、Mermaidコードブロックとして管理する。

# 最適と思われる理由
- コードと図表の同期が取れ、TypeScript定義変更時に差分を追いやすい。

# 入出力
- 入力: TypeScriptモジュールのフロー定義。
- 出力: Mermaid形式のUMLコード。

# 実装の評価
- Mermaid対応のエディタやCIプレビューで図が正しく描画されること。
