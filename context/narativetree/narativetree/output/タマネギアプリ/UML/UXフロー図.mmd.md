# 概要
`output/タマネギアプリ/UML/UXフロー図.mmd`は、玉ねぎアプリのユーザー体験をホーム導線別に可視化したMermaid活動図。

# 目的
- コンセプト・機能図と並べて参照できるよう、UMLディレクトリにUXフロー図を集約する。
- ホームから育てる/剥がす/フォロワーへの分岐とリフレクションまでの戻りを一目で確認する。

# 最適な実装方法
1. 既存の活動図コードを転記し、ディレクトリを整理。
2. Mermaid `stateDiagram-v2`を利用して状態遷移と注釈を記述。

# 最適と思われる理由
- 3図セットでレビューできるようになるため、コンセプト→機能→体験の流れが説明しやすい。

# 入出力
- 入力: UX設計メモ、既存Mermaidコード。
- 出力: UMLディレクトリに配置された活動図コード。

# 実装の評価
- コンセプト図・機能図と横並びで参照した際に矛盾がないかをレビューし、今後の変更時にも一括更新できること。
